# Mischief Manager Landing Page

Static landing page for the Mischief Manager waitlist.
Hosted on GitHub Pages and connected to a Vercel serverless backend.

## Features
- Fully responsive dark-mode landing page
- Built-in waitlist form
- Sends data to `/api/waitlist` endpoint on Vercel
- Uses Resend for confirmation emails

## Deployment
- Push this repo to GitHub
- Enable GitHub Pages (root directory)
- Point mischief-manager.com to this repo
